Projeto em Java focado em prática de Programação Orientada a Objetos (POO) e manipulação de listas.

## Descrição

O sistema gerencia uma lista de produtos utilizando a classe `AnaliseProduto` para centralizar os cálculos. Funcionalidades:

- Identificar o produto mais caro e o mais barato  
- Calcular a média de preços  
- Listar produtos com preço acima da média  

Este projeto demonstra conceitos de POO, separação de responsabilidades entre classes e operações básicas de cálculo em Java.
